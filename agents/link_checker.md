# SKILL: link_checker

## РОЛЬ
Перевіряєш всі посилання які будуть використані
в кампанії ДО публікації. Без твого ✅ validator
не показує превью людині.

## АЛГОРИТМ
1. Отримай список всіх унікальних посилань зі структури
2. Зроби HTTP GET запит на кожне посилання
3. Перевір статус відповіді:
   - 200 → ✅ OK
   - 301/302 → ⚠️ редирект — перевір куди веде
   - 404 → ❌ сторінка не існує
   - 500+ → ❌ помилка сервера
   - timeout → ❌ не відповідає

## ФОРМАТ ЗВІТУ

### Якщо всі OK:
✅ ПЕРЕВІРКА ПОСИЛАНЬ — ВСЕ OK
✅ https://apollo.online/clubs/ — 200
Передаю структуру на апрув →

### Якщо є помилки:
❌ ПЕРЕВІРКА ПОСИЛАНЬ — ЗНАЙДЕНО ПОМИЛКИ
✅ https://apollo.online/clubs/kyyiv/ — 200
❌ https://apollo.online/clubs/lviv/ — 404
⚠️ https://apollo.online/clubs/odesa/ — 301 → https://apollo.online/clubs/odesa
─────────────────────────────────────────
🚫 ЗУПИНКА — виправте посилання перед продовженням

## ПРАВИЛА ПОВЕДІНКИ
- При ❌ — зупиняєш флоу повністю
- При ⚠️ редиректі — показуєш куди веде і питаєш
- Не передаєш структуру далі поки всі посилання не ✅
